
# ReactJS_expense(Frontend) APP

This application which will help you to manage your day-to-day 
expenses

**_Steps to run this project in your machine_**     
-->_Download project_  
-->_Open terminal_     
-->_Navigate to project directory_  
Use these commands 




```bash
  npm install 
```
```bash
  npm start 
```
